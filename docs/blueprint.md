# **App Name**: Zenergy

## Core Features:

- Contract Management: Create and manage client contracts with different billing schedules.
- Scheduled Invoicing: Generate invoices based on predefined schedules (e.g., quarterly, end of term).
- P1 Invoice Management: Handle 'P1' invoices based on meter readings for services like hot water, heating, and fixed subscriptions with management fees.
- Meter Reading Input: Allows manual input of meter readings for each contract.
- PDF Invoice Generation: Generates invoices in PDF format, downloadable or emailable
- User access: App allows admin users and has access control.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to evoke trust and reliability. It balances professionalism with approachability.
- Background color: Very light blue (#E3F2FD) to provide a clean, uncluttered backdrop that reduces eye strain.
- Accent color: A muted teal (#4DB6AC) for highlighting key actions and information, contrasting softly with the blue hues.
- Body and headline font: 'Inter' sans-serif font, providing a modern, clean, and readable interface suitable for both headings and body text.
- Use consistent, simple line icons related to billing, contracts, and energy for visual clarity.
- Maintain a clean and well-organized layout for intuitive navigation. Use clear sections for contracts, meter readings, and invoice generation.
- Subtle transitions and animations for interactive elements, such as form submissions and invoice generation, enhancing user feedback.
